title: practical git for everyday developers

learner outcomes:
- create repositories
- commit to projects with course and granular additions
- revert commits (with forward history)
- checkout branches
- merge, unmerge, and rebase branches
- create, push to, and pull from remote

external to course:
- install latest git on {@os}
- practical {@git_provider} for everyday developers
- practical git for open source collaboration (tags, releases, branch, fork, merge, pr, rebase, etc)
- debugging with git (reflog, bisect)
- supercharged github,  with the gh cli
- using git in VS Code
