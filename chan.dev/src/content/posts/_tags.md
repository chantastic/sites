# Tag Vocabulary

Controlled vocabulary for post tags. When tagging posts, use only these tags.
Add new tags only when no existing tag fits and the new tag would apply to 3+ posts.

## Content type

| Tag | Use for |
|-----|---------|
| `reference` | Cheat sheets, shortcut tables, lookup pages |
| `tutorial` | Step-by-step guides that build something |
| `log` | Running logs, year reviews, weekly notes |
| `seed` | Short posts that may grow into longer pieces |

## Topic — Technical

| Tag | Use for |
|-----|---------|
| `react` | React library, hooks, components, RSC, React conf |
| `javascript` | JS language features, modules, regex, ES syntax |
| `web` | HTML, CSS, browser APIs, general web dev |
| `css` | CSS-specific (selectors, methodologies, layout) |
| `astro` | Astro framework |
| `storybook` | Storybook, doc blocks, CSF, Chromatic |
| `workos` | WorkOS, AuthKit, RBAC |
| `auth` | Authentication/authorization (often with workos) |
| `vim` | Vim, neovim, LazyVim, modal editing |
| `git` | Git, GitHub |
| `cli` | Terminal, shell, readline, command-line tools |
| `markdown` | Markdown, MDX, remark plugins |
| `ai` | AI tools, LLMs, coding agents, Claude Code |
| `seo` | Structured data, sitemaps, meta tags |
| `code` | Programming philosophy, abstractions, patterns |

## Topic — Tools & Gear

| Tag | Use for |
|-----|---------|
| `tools` | Software tools, apps, services, setup notes |
| `mac` | macOS, Mac settings, Mac-specific tips |
| `vscode` | VS Code settings, extensions, profiles |
| `gear` | Physical equipment, purchases, hardware |
| `audio` | Microphones, interfaces, podcast gear |
| `video` | Cameras, editing, DaVinci Resolve, Screen Studio |

## Topic — Life

| Tag | Use for |
|-----|---------|
| `life` | Personal essays, reflections, philosophy |
| `family` | Kids (Rock, Ruby), Nellie, parents, grandparents |
| `career` | Work, salary, interviews, job roles, visibility |
| `faith` | Bible, spirituality, church, wrestling with God |
| `parenting` | Parenting-specific (overlap with family is fine) |
| `speaking` | Conferences, workshops, emceeing |
| `content` | Content creation, editing, publishing, strategy |

## Legacy tags (do not add to new posts)

| Tag | Meaning |
|-----|---------|
| `found` | Captured thought or quote (use `seed` for new posts) |
| `note` | Raw note (use `seed` for new posts) |
| `from:pi` | Generated during a Pi session |
| `mode` | Day-of-week schedule posts |
| `play` | 2022 theme-related |
